# 🚀 ExamEdge - India's Premium JEE & NEET Hub
ExamEdge is a modern, responsive web platform designed specifically for JEE and NEET aspirants. It provides a seamless interface to access study materials, track syllabus progress, and manage academic resources effectively.

## ✨ Features
* **Premium Dashboard:** A personalized space for students to manage their subjects.
* **All-Device Responsive:** Works perfectly on Mobile, Tablet, and Desktop.
* **Modern UI/UX:** Clean design with smooth animations and glassmorphism effects.
* **Subject Tracking:** Dedicated sections for Physics, Chemistry, and Biology.
* **Interactive PDF Viewer:** Seamlessly integrated to view syllabus and notes.

## 🛠️ Tech Stack
* **HTML5:** Structure and content.
* **CSS3:** Custom styling and responsive layouts (Media Queries).
* **JavaScript:** Interactive elements and navigation logic.
* **FontAwesome:** High-quality icons.
* **Google Fonts:** Premium 'Poppins' typography.

## 📂 Project Structure
- `index.html` - The landing and login/signup page.
- `pages/` - Contains dashboard and subject-specific pages.
- `assets/` - Images, logos, and static resources.
- `css/` - Custom stylesheets.
- `js/` - Authentication and UI logic.

## 🚀 How to Run
1. Clone the repository or download the ZIP.
2. Open `index.html` in any modern web browser.
3. Use the login button to access the student dashboard.
